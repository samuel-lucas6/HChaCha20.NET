# HChaCha20.NET

A .NET implementation of [HChaCha20](https://datatracker.ietf.org/doc/html/draft-irtf-cfrg-xchacha#section-2.2).

> **Warning**
> 
> HChaCha20 is **NOT** a hash function and **MUST NOT** be used to derive keys from passwords.
